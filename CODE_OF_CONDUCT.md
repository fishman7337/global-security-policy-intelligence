# Code of Conduct

## Our Pledge

This project studies terrorism, governance, public policy, and security
analytics. Contributors are expected to treat the subject matter, affected
communities, and one another with care.

We pledge to make participation in this project respectful, evidence-driven,
and welcoming regardless of age, body size, caste, disability, ethnicity, sex
characteristics, gender identity and expression, level of experience,
education, nationality, personal appearance, race, religion, sexual identity
and orientation, socioeconomic status, or technology choices.

## Expected Behavior

- Use welcoming and inclusive language.
- Discuss sensitive security and violence topics in aggregate, research-safe
  terms.
- Give and receive feedback with humility, specificity, and respect.
- Credit data providers, prior research, and contributors.
- Prefer reproducible evidence over speculation.
- Respect project safety boundaries and avoid operational guidance.

## Unacceptable Behavior

- Harassment, threats, intimidation, stalking, or sustained disruption.
- Demeaning or discriminatory comments.
- Publishing private information without explicit permission.
- Using the project to request, contribute, or amplify tactical, targeting,
  weaponization, evasion, or operational guidance.
- Misrepresenting model outputs as definitive predictions of future violence.
- Ignoring dataset terms, citation requirements, or privacy constraints.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributions that
violate this Code of Conduct. Maintainers may also temporarily or permanently
limit participation when behavior creates safety, legal, or community risk.

Reports can be made through GitHub issues if safe to do so, or by contacting
the repository owner privately through GitHub. Reports should include links,
screenshots, or concrete context when possible.

## Scope

This code applies in all project spaces, including GitHub issues, pull
requests, discussions, documentation, notebooks, demos, and any community
spaces that clearly represent the project.
