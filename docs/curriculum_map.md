# Curriculum Map

This project intentionally maps DAAA modules into one integrated artifact.

| Module Area | Capstone Evidence |
|---|---|
| Programming Methodology | Typed Python modules, CLI scripts, reusable functions, tests |
| Programming for Data Analytics | `notebooks/01_data_engineering_quality.ipynb`, `notebooks/02_eda_visual_analytics.ipynb`, pandas EDA, Statsmodels-ready forecasting, feature engineering |
| Data Fluency | `notebooks/00_project_overview.ipynb`, `notebooks/01_data_engineering_quality.ipynb`, summary statistics, missingness checks, business-style KPIs |
| Data Visualisation | `notebooks/02_eda_visual_analytics.ipynb`, `notebooks/03_geospatial_forecasting.ipynb`, React dashboard, Plotly charts, geospatial aggregate maps |
| Persuasive Communication | Narrative dashboard, README, methodology and ethics docs |
| Mathematics / Quantitative AI | `notebooks/04_adaptive_severity.ipynb`, `notebooks/10_policy_research_panel.ipynb`, probability, uncertainty bands, calibration-ready model evaluation |
| AI & Machine Learning | `notebooks/05_feature_store_ml_baselines.ipynb`, `notebooks/06_ml_experiment_benchmark.ipynb`, classification, regression plan, clustering, chatbot, explainability |
| Deep Learning | `notebooks/07_deep_learning_lab.ipynb`, Keras MLP and sequence-window scaffold for forecasting |
| Generative AI with LLMs | `notebooks/09_rag_mcp_safety.ipynb`, RAG chatbot, prompt safety, MCP tools, retrieval evaluation |
| Back-End Web Development | FastAPI database-driven API contracts |
| Front-End Web Development | Responsive React/Vite application |
| Data Engineering | `notebooks/01_data_engineering_quality.ipynb`, bronze/silver/gold pipelines, Parquet, PostGIS schema |
| Big Data Technologies | `notebooks/01_data_engineering_quality.ipynb`, Spark-ready pipeline with pandas fallback and Parquet lake layout |
| Data Structures & Algorithms | Hash maps, heaps, queues, union-find, graph traversal, complexity report |
| Graph Analytics | `notebooks/08_graph_analytics.ipynb`, Neo4j schema exports, centrality, PageRank baseline, communities |
| Agile MLOps | `notebooks/11_monitoring_drift_mlops.ipynb`, W&B tracking, artifacts, Docker, CI tests, reproducible commands |
| Computer Organisation | CLI scripts, containers, networking, service health checks |

The project goes beyond curriculum with RAG safety, MCP integration, graph database design, model cards, CI/CD, and production-style service boundaries.
