# Support

This is a research and portfolio project, not an operational security service.

## Getting Help

Use GitHub issues for:

- Setup problems.
- Reproducibility questions.
- Documentation gaps.
- API, notebook, frontend, or pipeline bugs.
- Public-policy methodology questions.

Please include your operating system, Python version, Node version, command
output, and the smallest reproduction you can provide.

## Not Supported

The project does not provide:

- Tactical or operational security advice.
- Predictions about specific future incidents.
- Targeting, threat actor profiling, or enforcement guidance.
- Assistance bypassing dataset access rules.
- Emergency response support.

For immediate safety emergencies, contact appropriate local authorities or
qualified emergency services.
