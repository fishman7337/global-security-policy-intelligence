# Notice

Global Security Policy Intelligence is licensed under Apache-2.0 for code and
project-authored documentation unless another license is explicitly stated.

This repository is designed to work with third-party datasets that have their
own access rules, citation requirements, and licensing terms. Those datasets
are not relicensed by this project.

Important source families include:

- Global Terrorism Database (GTD), START, University of Maryland.
- Worldwide Governance Indicators (WGI), World Bank.
- World Development Indicators (WDI), World Bank.
- V-Dem dataset and documentation.
- UNDP Human Development Index data.
- UCDP organized violence datasets.
- Freedom House Freedom in the World.
- International IDEA Global State of Democracy Indices.

Raw restricted or bulky datasets should not be committed to this repository.
Use documented source registries and local artifact generation instead.
